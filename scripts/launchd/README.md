# Launchd Assets

This directory stores machine-specific launchd wrappers and plist files generated by `./scripts/install-local.sh`.

Tracked source files should not include absolute paths, local usernames, or generated launchd artifacts.

Expected generated files:

- `cortex-serve.sh`
- `ilink-agent.sh`
- `com.cortex.serve.plist`
- `com.cortex.ilink-agent.plist`
- `com.cortex.digest-cron.plist`

If they are missing, rerun `./scripts/install-local.sh` before using `./scripts/cortex-services.sh`.
